# @extx/utils

一些常用的JS工具。

## 安装

使用 pnpm 安装：

```bash
pnpm add @extx/utils
```

## 使用方法

```javascript
import { isNumber } from '@extx/utils'

// 示例代码
const isNum1 = isNumber(1)
const isNum2 = isNumber('1')

```

## 功能

- **isNumber**: 是否为Number类型
- **isString**: 是否为String类型
